# OIBSIP_Basic-Firewall-Configuration-with-UFW

## Objective
Configure a basic firewall on a Linux system using UFW to allow SSH and block HTTP traffic.

## Tools Used
- UFW (Uncomplicated Firewall)
- Kali Linux (VM)

## Configuration Steps
1. Installed UFW
2. Allowed SSH traffic (port 22)
3.  Denied HTTP traffic (port 80)
4.  Enabled the firewall
5.  Verified active rules using `ufw status`

## Result
The system is protected by UFW with SSH access allowed and HTTP traffic blocked.
